== Installation ==
1. Install and Activate mycred 1.5.x or higher.
2. Upload `myCRED_Zarinpal` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Enjoy It :)
